# Random Matches

A BinarySearch client that takes an int value T as command-line argument 
and runs T trials of the following experiment for N = 10³, 10⁴, 10⁵, and 10⁶:
it generate two arrays of N randomly generated positive six-digit int values, 
and finds the number of values that appear in both arrays.
Then Prints a table giving the average value of this quantity over the T trials
for each value of N.
